# MCP Browser Use Server

Browser automation capabilities via the Model Context Protocol. This service is planned but not yet deployed.

## Status

**Planned** — configuration directory reserved. No Docker service is currently defined in `docker-compose.yaml`.

## Purpose

When deployed, this service will provide browser automation tools (navigation, clicking, form filling, content extraction) accessible to AI models through MCP.
